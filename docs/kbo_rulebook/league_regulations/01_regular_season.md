# KBO 정규시즌 규정

## 개요
KBO 정규시즌의 기본적인 경기 규정과 리그 운영 원칙을 다룹니다.

## 세부 조항

### 조항 1: 리그 구성
- **정규시즌**: 10개 팀이 144경기씩 진행
- **경기 일정**: 3월 말 ~ 10월 초 (약 6개월)
- **홈/원정**: 각 팀당 72경기씩 홈경기와 원정경기
- **더블헤더**: 우천 등으로 인한 연기 경기 소화를 위해 실시

### 조항 2: 경기 시간
- **정규 이닝**: 9이닝
- **연장전**: 12이닝까지 (2021년부터 적용)
- **콜드게임**: 5이닝 이상 진행 후 7점차 이상 시 적용
- **경기 시작**: 평일 18:30, 주말/공휴일 14:00 또는 17:00

### 조항 3: 타이브레이크 제도
- **적용 시점**: 10이닝부터 무사 1,2루 상황에서 시작
- **목적**: 연장전 장기화 방지
- **적용 방법**: 전 이닝 마지막 두 타자가 2루, 1루에 출루한 상태로 시작

### 조항 4: 지명타자(DH) 제도
- **적용**: 모든 경기에서 투수를 대신하여 지명타자 사용 가능
- **교체**: 지명타자가 수비에 나가면 투수가 타순에 들어감
- **번호**: 타순에서 지명타자는 'D' 또는 '지'로 표기

### 조항 5: 비디오 판독 제도
- **도입**: 2014년부터 시행
- **대상 판정**: 홈런, 방해, 포스/태그 플레이 등
- **신청**: 각 팀당 3회까지 신청 가능
- **제한 시간**: 판독 시간 최대 90초

### 조항 6: 순위 결정
- **순위 기준**: 팀 순위는 승률에 따라 결정된다.
- **승률 계산**: 승률 = 총 승수 / (총 승수 + 총 패수). 무승부는 승률 계산에서 제외한다.
- **게임 차(GB)**: 1위 팀과의 승패 차이를 나타내는 지표로, 순위 결정의 직접적인 기준은 아니다.

### 조항 7: 동률 순위 처리
- **정규시즌 최종 순위 동률**: 두 팀 이상이 동률일 경우, 다음 순서에 따라 순위를 결정한다.
    1. 동률 팀 간의 상대 전적에서 우위인 팀
    2. 동률 팀 간의 다득점 순위가 높은 팀
    3. 전 시즌 순위가 높았던 팀
- **포스트시즌 진출 동률**: 포스트시즌 진출이 걸린 순위(예: 5위)에서 동률이 발생할 경우, 별도의 순위 결정전(타이브레이커 경기)을 단판으로 치른다.

### 조항 8: 경기 취소 및 중단
- **노 게임(No Game)**: 5이닝을 마치기 전에 경기가 중단되고 재개될 수 없을 경우 '노 게임'으로 선언되며, 해당 경기의 모든 기록은 무효가 되고 추후 재편성된다.
- **서스펜디드 게임(Suspended Game)**: 5이닝을 마친 후 동점인 상황에서 경기가 중단될 경우 '일시 중단 경기'로 선언되며, 추후 중단된 시점부터 경기를 재개한다.
- **무승부 처리**: 연장 12이닝까지 승패를 가리지 못할 경우 무승부로 처리하며, 해당 경기는 재편성하지 않는다.

## 예시

### 타이브레이크 적용 사례
```
10회초: 무사 1,2루 상황에서 공격 시작
- 1루 주자: 전 이닝 8번 타자
- 2루 주자: 전 이닝 9번 타자
- 타석: 1번 타자부터 시작
```

### 콜드게임 적용 사례
```
5회말 경기 중 7점 이상 차이가 나고 비가 내려 경기 중단
→ 선두팀 승리로 게임 종료
```

### 순위 결정 예시
```
A팀: 80승 60패 4무 → 승률 0.571
B팀: 78승 58패 8무 → 승률 0.573
→ B팀의 순위가 더 높다 (무승부는 계산에서 제외)
```

### 경기 취소 사례
```
3회초 경기 중 폭우로 중단, 1시간 대기 후에도 재개 불가
→ 심판진 '노 게임' 선언
→ 해당 경기는 추후 재편성, 모든 기록 무효
```
