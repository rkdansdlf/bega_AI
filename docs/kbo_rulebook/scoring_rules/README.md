# KBO 공식 기록 규칙

## 개요
경기 기록, 통계 등 공식 기록에 관한 규정을 다룹니다.

## 세부 조항

### 조항 1: 기록 규정
- **공식 기록**: KBO 공인 기록원이 작성한 기록만 공식 인정.
- **기록 정정**: 경기 종료 후 24시간 이내 정정 신청 가능. 구단은 정정 요청 사유를 명시하여 KBO 기록위원회에 공식 제출하며, 위원회는 영상 분석 등을 통해 최종 결정을 내린다.
- **기록 공시**: 최종 확정된 기록은 KBO 공식 홈페이지를 통해 공시되며, 이는 영구 보관된다.

### 조항 2: 타격 기록 세칙
- **안타(Hit)**: 타자가 수비의 실책이나 야수 선택 없이 안전하게 1루 이상에 도달한 타구.
- **실책(Error)**: 평범한 수비로 처리했어야 할 타구를 야수가 놓치거나, 잘못된 송구로 타자 또는 주자를 더 진루시킨 경우.
- **야수 선택(Fielder's Choice)**: 타자의 땅볼 타구에 수비수가 타자 주자 대신 선행 주자를 아웃시키려고 시도했거나, 선행 주자를 아웃시킨 경우. 이 때 타자에게는 안타를 주지 않는다.
- **희생 번트(Sacrifice Bunt)**: 무사 또는 1사 상황에서 타자가 번트를 통해 주자를 진루시키고 자신은 아웃된 경우. 타자의 타수에는 포함되지 않는다.
- **희생 플라이(Sacrifice Fly)**: 무사 또는 1사, 3루 주자가 있는 상황에서 타자의 외야 뜬공으로 3루 주자가 득점한 경우. 타자의 타수에는 포함되지 않는다.

### 조항 3: 투구 기록 세칙
- **승리 투수**:
    - 선발 투수: 최소 5이닝을 투구하고, 교체될 때까지 팀이 리드를 유지하여 경기가 종료된 경우.
    - 구원 투수: 팀이 동점 또는 역전 상황일 때 등판하여, 투구 중 팀이 결승점을 얻고 그 리드를 지켜 경기가 끝났을 경우.
- **패전 투수**: 투수가 허용한 주자가 결승점이 되어 팀이 패배했을 경우.
- **세이브(Save)**: 리드 상황에서 등판하여 경기를 마무리한 구원 투수에게 주어지며, 아래 조건 중 하나를 만족해야 한다.
    1. 3점 이하의 리드에서 1이닝 이상 투구.
    2. 베이스에 있는 주자, 상대하는 타자, 또는 그 다음 타자에게 홈런을 맞으면 동점 또는 역전이 되는 상황에서 등판.
- **홀드(Hold)**: 세이브와 유사한 조건에서 등판하여 리드를 지키고 다음 투수에게 마운드를 넘긴 구원 투수에게 주어진다.
- **자책점(Earned Run)**: 투수가 실책이나 패스트볼의 도움 없이 자신의 투구로 인해 허용한 실점. 실책이 없었다면 이닝이 종료되었을 상황 이후의 모든 실점은 비자책점으로 간주한다.

### 조항 4: 통계 산출 규정
- **타율**: 안타수 ÷ 타수
- **출루율**: (안타+볼넷+몸에 맞는 공) ÷ (타수+볼넷+몸에 맞는 공+희생플라이)
- **장타율**: 총 루타수 ÷ 타수
- **방어율(ERA)**: (자책점 × 9) ÷ 투구 이닝수

## 예시

### 승리 투수 판정 사례
```
선발투수: 5이닝 3실점으로 강판 (팀이 5-3 리드)
→ 이후 팀이 계속 리드 유지하여 승리
→ 선발투수가 승리투수

구원투수: 8회에 등판하여 동점 상황을 만듦
→ 9회에 팀이 결승점 득점하여 승리
→ 구원투수가 승리투수
```

### 자책점/비자책점 사례
```
2사 주자 없는 상황에서 유격수 실책으로 주자 출루.
→ 다음 타자가 홈런을 침 (2실점)
→ 실책이 없었다면 이닝이 종료되었을 것이므로, 홈런으로 인한 2실점 모두 비자책점으로 기록.
```

### 야수 선택 사례
```
1사 1루, 타자가 유격수 앞 땅볼을 침.
→ 유격수가 2루로 뛰는 주자를 아웃시키고, 타자 주자는 1루에서 세이프.
→ 타자에게는 안타가 아닌 '야수 선택'으로 기록.
```
