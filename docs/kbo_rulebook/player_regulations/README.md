# KBO 선수 규정

## 개요
선수 등록, 자격, 계약, 이적 등 선수와 관련된 모든 규정을 다룹니다.

## 세부 조항

### 조항 1: 선수 등록
- **엔트리**: 40명 (1군 28명 + 2군 12명)
- **출전 엔트리**: 경기당 25명 (투수 13명, 야수 12명)
- **외국인 선수**: 최대 3명 등록, 2명까지 동시 출전
- **등록 기간**: 시즌 중 자유롭게 등록/말소 가능

### 조항 2: 외국인 선수 규정
- **등록 제한**: 팀당 3명까지 등록, 2명까지 동시 출전
- **포지션 제한**: 포수는 외국인 선수 등록 불가
- **계약 조건**: 1년 단위 계약, 최대 연봉 상한선 적용
- **FA 자격**: 외국인 선수는 FA 자격 없음

### 조항 3: 자유계약선수(FA) 제도
- **자격 요건**: 8시즌 이상 1군 등록 (최소 145일/시즌)
- **보상 선수**: FA 영입 팀은 원 소속팀에 보상 제공
- **등급 분류**: A급, B급, C급으로 분류하여 보상 차등화
- **행사 기간**: 시즌 종료 후 15일 이내 FA 자격 행사

### 조항 4: 신인 드래프트
- **연령 제한**: 고등학교 졸업 예정자, 대학교 4학년
- **지명 순서**: 전년도 성적 역순으로 지명
- **계약 조건**: 신인 최저연봉 적용, 계약금 상한선 존재
- **입단 거부**: 지명 거부 시 다음 해 재지명 가능

### 조항 5: 트레이드 제도
- **기간 제한**: 시즌 중 특정 기간 내에만 가능
- **승인 절차**: KBO 사무국의 승인 필요
- **선수 동의**: 10년 이상 선수는 거부권 보유
- **금전 트레이드**: 순수 금전 거래는 금지

### 조항 6: 웨이버 공시
- **대상**: 방출 예정 선수를 다른 팀에 우선 공개
- **기간**: 72시간 동안 웨이버 공시
- **순서**: 성적 역순으로 획득 우선권
- **조건**: 기존 계약 조건 그대로 승계

### 조항 7: 계약 및 연봉
- **선수 계약**: 모든 선수는 KBO 표준 계약서에 따라 구단과 계약을 체결해야 한다.
- **최저 연봉**: KBO 리그에 등록된 모든 선수는 매년 이사회에서 정하는 최저 연봉 이상을 보장받는다.
- **연봉 조정**: 특정 자격을 갖춘 선수(3시즌 이상 등록)가 구단과 연봉 합의에 이르지 못할 경우, KBO 연봉조정위원회에 조정을 신청할 수 있다.
- **보류권**: 구단은 매년 11월 30일까지 다음 시즌에 함께할 선수를 '보류선수'로 지정하여 명단을 공시하며, 해당 선수에 대한 독점 교섭권을 갖는다.

### 조항 8: 부상자 명단 (IL)
- **등재**: 선수가 부상으로 인해 경기에 출전할 수 없을 때, 구단은 해당 선수를 부상자 명단에 등재할 수 있다.
- **최소 기간**: 부상자 명단에 등재된 선수는 최소 10일, 15일 또는 60일(기간에 따라 종류 상이) 동안 1군 경기에 출전할 수 없다.
- **복귀**: 최소 등재 기간이 지나고, 선수가 부상에서 회복되면 1군 엔트리에 다시 등록될 수 있다.
- **엔트리**: 부상자 명단에 등재된 선수는 1군 엔트리에서 제외되므로, 다른 선수를 등록할 수 있다.

### 조항 9: 특수 신분 선수
- **군 보류 선수**: 병역 의무를 이행하는 선수는 '군 보류 선수'로 공시되며, 복무 기간 동안 계약 관계가 유지된다. 전역 후에는 원소속팀으로 자동 복귀한다.
- **육성 선수**: 정식 선수 등록(40명 엔트리)에 포함되지 않은 선수로, 2군(퓨처스리그) 경기에만 출전할 수 있다. 시즌 중 특정 기간에 정식 선수로 전환될 수 있다.
- **임의 해지 선수**: 선수가 계약 기간 중 개인 사정 등으로 구단의 동의 하에 활동 중단을 요청할 경우 '임의 해지 선수'로 공시된다. 공시 후 1년이 지나야 구단의 동의 하에 복귀할 수 있다.

## 예시

### FA 보상 등급별 기준
```
A급 (연봉 6억 이상): 1라운드 지명권 + 보상선수
B급 (연봉 3-6억): 2라운드 지명권 + 보상선수  
C급 (연봉 3억 미만): 보상선수만
```

### 외국인 선수 동시 출전 제한
```
등록: 투수 2명, 야수 1명 (총 3명)
출전: 투수 1명, 야수 1명 (총 2명까지)
포수 외국인은 등록 불가
```

### 드래프트 지명 순서 (2024년 기준)
```
1순위: SSG 랜더스 (10위)
2순위: NC 다이노스 (9위)
3순위: 한화 이글스 (8위)
...
10순위: KIA 타이거즈 (1위)
```

### 연봉 조정 신청 사례
```
프로 4년차 선수가 구단과 연봉 협상에 실패
→ 선수가 KBO에 연봉 조정 신청
→ 조정위원회에서 구단 제시액과 선수 요구액을 바탕으로 최종 연봉 결정
```

### 부상자 명단 등재
```
선발 투수가 어깨 부상으로 10일 이상 결장 예상
→ 구단은 해당 선수를 '10일 부상자 명단'에 등재
→ 엔트리에서 제외하고, 2군에서 다른 투수를 등록
```
