# KBO 선수 규정

## 개요
선수 등록, 자격, 계약, 이적 등 선수와 관련된 모든 규정을 다룹니다.

## 세부 조항

### 조항 1: 선수 등록
- **선수 등록 현황**: KBO 공식 홈페이지는 구단별 선수 등록 현황과 퓨처스 선수 등록 현황을 별도 공개한다.
- **엔트리 등록 일수**: KBO 기록실 선수 페이지에는 선수별 엔트리 등록 일수와 말소 일수가 표시된다.
- **등·말소**: 시즌 중 선수 등록과 말소는 공식 등록 현황 및 선수 기록 페이지에서 확인한다.
- **세부 숫자 기준**: 등록 인원, 출전 인원, 시즌 중 운영 기준은 시즌별 KBO 규정·공시를 우선 확인해야 한다.

### 조항 2: 외국인 선수 규정
- **운영 기준**: 외국인 선수 등록과 출전 기준은 시즌별 KBO 규정 변경 공지와 리그 운영 규정을 우선 확인해야 한다.
- **공식 확인 경로**: 세부 제한은 KBO의 시즌별 규정·규칙 변화 공지와 규정・자료실 문서를 기준으로 본다.

### 조항 3: 자유계약선수(FA) 제도
- **FA 제도 존재**: 자유계약선수(FA) 제도와 보상선수 기준은 KBO 선수 관련 규정에서 다루는 핵심 주제다.
- **보상 규정 확인**: 보상선수 여부, 보상 방식, 자격 기준은 시즌별 공식 규정과 공시를 기준으로 확인해야 한다.
- **주의**: 연차, 등록일수, 등급별 보상 기준 등 숫자 규정은 시즌별 개정 가능성이 있어 공식 문서 대조 없이 단정하지 않는다.

### 조항 4: 부상자 명단 (IL)
- **등재/말소 흐름**: 부상자 명단 관련 이동은 선수별 엔트리 등록 일수·말소 일수 기록과 공식 등록 현황에서 확인할 수 있다.
- **복귀**: 부상 상태 해소 후 엔트리 재등록 여부는 공식 엔트리 변동 내역을 기준으로 확인한다.
- **주의**: 최소 일수와 세부 분류는 시즌별 운영 규정 변경 가능성이 있어 최신 공식 규정을 우선한다.

### 조항 5: 특수 신분 선수
- **군 보류 선수**: 병역 의무를 이행하는 선수는 '군 보류 선수'로 공시되며, 복무 기간 동안 계약 관계가 유지된다. 전역 후에는 원소속팀으로 자동 복귀한다.
- **육성 선수**: 육성 선수와 정식 선수 전환 여부는 시즌별 선수 등록 규정과 구단 공시를 기준으로 확인한다.
- **임의 해지 선수**: 선수가 계약 기간 중 개인 사정 등으로 구단의 동의 하에 활동 중단을 요청할 경우 '임의 해지 선수'로 공시된다. 공시 후 1년이 지나야 구단의 동의 하에 복귀할 수 있다.

## 예시

### FA/보상선수 확인 포인트
```
- 자유계약선수(FA) 자격 조건
- 보상선수 대상 여부
- 등급별 보상 방식
- 시즌별 개정 사항 존재 여부
```

### 엔트리·부상자 명단 확인 예시
```
선수 기록 페이지에서 엔트리 등록 일수 / 말소 일수 확인
→ 선수 등록 현황 페이지에서 최근 등·말소 내역 확인
→ 최신 규정 수치는 시즌 공지와 규정・자료실 원문으로 재대조
```
