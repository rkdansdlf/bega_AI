# 2025 KBO 시즌 스토리라인 해설

이 문서는 2025 시즌형 질문을 분류하고 답변 범위를 제한하기 위한 운영용 정적 문서입니다. 순위, 선수 성적, 경기 결과, 부상, 엔트리, 수상, 우승 여부처럼 정확한 기록이 필요한 항목은 내부 DB 또는 운영자 검수 데이터가 있을 때만 답변해야 합니다.

## 시즌 질문 분류

2025 시즌 질문은 보통 팀 흐름, 선수 기록, 순위 경쟁, 포스트시즌 가능성, 신인 활약, 외국인 선수, 불펜 운용, 타선 생산력, 수비 안정성, 구장 경험으로 나뉩니다. 답변할 때는 사용자가 묻는 축이 팀인지 선수인지, 누적 기록인지 최근 흐름인지, 정규시즌인지 포스트시즌인지 먼저 구분합니다.

## 답변 원칙

- 수치가 필요한 질문은 내부 DB 근거가 없으면 단정하지 않습니다.
- “요약해줘”, “흐름을 알려줘” 같은 질문은 기준 연도, 팀명, 선수명, 경기일을 좁히도록 유도합니다.
- 기록 비교는 같은 시즌, 같은 리그 범위, 같은 포지션 또는 같은 통계 기준으로만 비교합니다.
- 최신성이나 운영자 검수가 필요한 질문은 `MANUAL_BASEBALL_DATA_REQUIRED` 계약을 우선합니다.

## 안전한 일반 안내

데이터가 없는 상태에서도 제공할 수 있는 답변은 질문을 좁히는 방법, 확인해야 할 데이터 종류, 비교 기준 안내로 제한합니다. 예를 들어 팀 흐름 질문에는 승패, 득실, 선발진, 불펜, 타선, 수비, 홈/원정 차이를 같은 기준일로 확인해야 한다고 안내할 수 있습니다.
