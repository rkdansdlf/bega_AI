# KBO 구단, 구장, 팬문화, 역사 해설

이 문서는 KBO 챗봇이 구단 문화, 구장 경험, 응원 문화, 역사형 질문을 분류하고 답변할 때 사용하는 정적 지식 문서입니다. 시즌별 순위, 선수 이동, 우승 횟수, 최신 응원가, 당일 경기 정보처럼 변동 가능한 내용은 내부 DB 또는 운영자 제공 데이터가 있을 때만 답변해야 합니다.

## 팬문화와 응원

KBO 팬문화는 홈구장 응원석, 팀별 응원 구호, 선수별 등장곡, 치어리딩, 원정 응원 문화가 함께 결합된 형태로 이해할 수 있습니다. 응원 문화 질문에는 특정 시즌의 최신 곡명이나 가사를 단정하지 말고, 응원 방식과 관람 예절을 중심으로 안내합니다.

응원가나 구호를 묻는 질문은 저작권과 최신성 이슈가 있으므로 전체 가사나 정확한 최신 목록을 생성하지 않습니다. 필요한 경우 앱의 공식 구단 링크, 내부에 저장된 운영자 검수 데이터, 또는 구장 안내 데이터를 확인하도록 유도합니다.

## 구장 경험

구장 경험 질문은 좌석 시야, 이동 동선, 매점, 화장실, 주차, 대중교통, 우천 대응, 원정 팬 좌석, 가족 관람 편의처럼 관람자가 실제로 확인해야 하는 정보가 많습니다. 장소별 운영 시간, 예매처, 가격, 주차 가능 여부는 변동될 수 있으므로 내부 장소 DB나 운영자 제공 데이터를 우선합니다.

일반 안내가 가능한 경우에는 준비물, 도착 시간, 날씨 대비, 음식물 반입 확인, 경기 종료 후 이동 혼잡 같은 실용적인 범위에서 답변합니다.

## 구단 역사 질문

구단 역사 질문은 창단, 연고지, 홈구장, 상징색, 라이벌 구도, 팬덤 분위기, 주요 전성기 같은 주제로 나뉩니다. 다만 우승 횟수, 시즌별 성적, 특정 선수의 소속 이력처럼 정확한 수치나 연도가 필요한 항목은 내부 DB 근거가 있을 때만 답변합니다.

역사형 질문에 DB 근거가 부족하면 “확인된 내부 자료 기준으로는 단정할 수 없습니다”라고 밝히고, 사용자가 팀명, 연도, 대회, 선수명 중 하나를 더 좁혀 질문하도록 안내합니다.

## 답변 원칙

- 변동 데이터는 내부 DB 또는 운영자 검수 데이터가 없으면 단정하지 않습니다.
- 응원가 가사는 생성하지 않고, 응원 문화의 맥락만 설명합니다.
- 구장 편의 정보는 실제 방문 전 공식 안내 또는 앱 내 장소 데이터를 확인하도록 안내합니다.
- 역사형 질문은 팀명, 시즌, 선수명, 사건명을 분리해 근거가 있는 범위에서 답변합니다.
