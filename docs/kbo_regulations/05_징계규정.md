# KBO 징계 규정

## 개요
선수, 코치, 구단의 위반 행위에 대한 징계 기준과 절차를 다룹니다.

## 세부 조항

### 조항 1: 징계 사유
- **폭력 행위**: 상대 선수, 심판, 관중에 대한 폭력
- **도박 행위**: 야구 경기 관련 도박 참여
- **약물 사용**: 금지 약물 복용 및 도핑
- **심판 모독**: 심판에 대한 과도한 항의 및 모독

### 조항 2: 징계 종류
- **경고**: 1차 위반 시 경고 조치
- **출장 정지**: 1경기~무기한 출장 정지
- **벌금**: 위반 정도에 따른 벌금 부과
- **자격 박탈**: 영구 제명 등 중징계

### 조항 3: 폭력 행위 징계
- **경미한 폭행**: 5~10경기 출장 정지
- **심각한 폭행**: 30경기 이상~무기한 정지
- **집단 폭행**: 관련자 모두 중징계
- **재범 시**: 이전 징계의 2배 이상 가중

### 조항 4: 도박 관련 징계
- **경기 결과 도박**: 영구 제명
- **일반 도박**: 30경기 이상 출장 정지
- **정보 제공**: 관련 정보 제공 시 중징계
- **은폐 시**: 사실 은폐 시 가중 처벌

### 조항 5: 약물 관련 징계
- **1차 적발**: 50경기 출장 정지
- **2차 적발**: 100경기 출장 정지
- **3차 적발**: 영구 제명
- **치료 프로그램**: 의무적 약물 치료 프로그램 이수

### 조항 6: 심판 모독 징계
- **언어적 모독**: 1~5경기 출장 정지
- **신체 접촉**: 10경기 이상 출장 정지
- **기물 파손**: 기물 배상 + 출장 정지
- **반복 시**: 누적 징계로 가중 처벌

### 조항 7: 징계 절차
- **조사 위원회**: 사실관계 조사 실시
- **소명 기회**: 당사자 소명 기회 제공
- **징계위원회**: 최종 징계 수위 결정
- **이의 신청**: 징계 결정에 대한 이의 신청 가능

### 조항 8: 구단 징계
- **관리 감독 소홀**: 구단에 벌금 부과
- **시설 관리**: 경기장 시설 문제 시 제재
- **팬 관리**: 응원단 문제 행동 시 구단 책임
- **규정 위반**: 각종 리그 규정 위반 시 제재

## 예시

### 폭력 행위 징계 사례
```
2019년 시즌: 선수 간 몸싸움 발생
→ 관련 선수들 5~10경기 출장 정지
→ 벌금 추가 부과

심판 폭행 사례: 심판에게 신체 접촉
→ 30경기 출장 정지 + 벌금 1000만원
```

### 도박 징계 사례
```
경기 승부 조작 관련 도박
→ 관련 선수 영구 제명
→ 구단도 연대 책임으로 제재

일반 도박장 출입
→ 30경기 출장 정지
→ 재범 시 가중 처벌
```

### 약물 사용 징계
```
1차 적발: 스테로이드 사용
→ 50경기 출장 정지
→ 의무 약물 치료 프로그램 이수

2차 적발: 동일 선수 재적발
→ 100경기 출장 정지
→ 강화된 관리 프로그램 적용
```

### 심판 모독 사례
```
경미한 항의: 볼 판정에 대한 과도한 항의
→ 경고 조치

심각한 모독: 심판에게 욕설 및 위협
→ 5경기 출장 정지 + 벌금
```
