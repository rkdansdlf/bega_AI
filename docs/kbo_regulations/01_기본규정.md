# KBO 기본 규정

## 개요
KBO 리그의 가장 기본적인 경기 규정과 리그 운영 원칙을 다룹니다.

## 세부 조항

### 조항 1: 리그 구성
- **정규시즌**: 10개 팀이 144경기씩 진행
- **경기 일정**: 3월 말 ~ 10월 초 (약 6개월)
- **홈/원정**: 각 팀당 72경기씩 홈경기와 원정경기
- **더블헤더**: 우천 등으로 인한 연기 경기 소화를 위해 실시

### 조항 2: 경기 시간
- **정규 이닝**: 9이닝
- **연장전**: 12이닝까지 (2021년부터 적용)
- **콜드게임**: 5이닝 이상 진행 후 7점차 이상 시 적용
- **경기 시작**: 평일 18:30, 주말/공휴일 14:00 또는 17:00

### 조항 3: 타이브레이크 제도
- **적용 시점**: 10이닝부터 무사 1,2루 상황에서 시작
- **목적**: 연장전 장기화 방지
- **적용 방법**: 전 이닝 마지막 두 타자가 2루, 1루에 출루한 상태로 시작

### 조항 4: 지명타자(DH) 제도
- **적용**: 모든 경기에서 투수를 대신하여 지명타자 사용 가능
- **교체**: 지명타자가 수비에 나가면 투수가 타순에 들어감
- **번호**: 타순에서 지명타자는 'D' 또는 '지'로 표기

### 조항 5: 비디오 판독 제도
- **도입**: 2014년부터 시행
- **대상 판정**: 홈런, 방해, 포스/태그 플레이 등
- **신청**: 각 팀당 3회까지 신청 가능
- **제한 시간**: 판독 시간 최대 90초

## 예시

### 타이브레이크 적용 사례
```
10회초: 무사 1,2루 상황에서 공격 시작
- 1루 주자: 전 이닝 8번 타자
- 2루 주자: 전 이닝 9번 타자
- 타석: 1번 타자부터 시작
```

### 콜드게임 적용 사례
```
5회말 경기 중 7점 이상 차이가 나고 비가 내려 경기 중단
→ 선두팀 승리로 게임 종료
```
