# KBO 포스트시즌 규정

## 개요
정규시즌 종료 후 진행되는 플레이오프와 한국시리즈에 관한 규정을 다룹니다.

## 세부 조항

### 조항 1: 포스트시즌 진출
- **진출 팀**: 정규시즌 상위 5개 팀
- **1위**: 한국시리즈 직행 (4승 어드밴티지)
- **2-3위**: 준플레이오프 (5전 3선승제)
- **4-5위**: 와일드카드 결정전 (1경기)

### 조항 2: 와일드카드 결정전
- **참가 팀**: 정규시즌 4위 vs 5위
- **경기 방식**: 단판 승부 (1경기)
- **홈 어드밴티지**: 정규시즌 순위 상위팀(4위)
- **승리팀**: 준플레이오프 진출

### 조항 3: 준플레이오프
- **참가 팀**: 정규시즌 2위 vs 와일드카드 승리팀, 3위팀
- **경기 방식**: 5전 3선승제
- **홈 어드밴티지**: 정규시즌 순위 상위팀
- **승리팀**: 플레이오프 진출

### 조항 4: 플레이오프
- **참가 팀**: 준플레이오프 승리팀 vs 정규시즌 1위팀
- **경기 방식**: 7전 4선승제
- **1위 어드밴티지**: 1승 선취 (4승 시스템)
- **승리팀**: 한국시리즈 진출

### 조항 5: 한국시리즈
- **참가 팀**: 플레이오프 승리팀 vs 정규시즌 1위팀(직행) 또는 플레이오프 승리 2팀
- **경기 방식**: 7전 4선승제
- **홈 어드밴티지**: 정규시즌 성적 우수팀
- **우승 팀**: KBO 리그 챔피언

### 조항 6: 특별 규정
- **연장전**: 정규시즌과 동일 (12이닝까지)
- **타이브레이크**: 10이닝부터 적용
- **지명타자**: 모든 경기에서 적용
- **엔트리**: 포스트시즌 전용 엔트리 28명 등록

### 조항 7: 경기장 및 일정
- **홈 경기**: 정규시즌 성적 우수팀이 홈 어드밴티지
- **경기 간격**: 최소 1일 간격 원칙
- **우천 연기**: 연기 시 익일 순연
- **중립 구장**: 특별한 경우가 아니면 사용하지 않음

### 조항 8: 개인 기록
- **포스트시즌 기록**: 정규시즌과 별도 관리
- **MVP 선정**: 각 라운드별 MVP 선정
- **한국시리즈 MVP**: 전체 포스트시즌 최고 선수
- **기록 인정**: 모든 개인 기록 공식 인정

## 예시

### 2024 포스트시즌 진출 (가상)
```
1위: KIA 타이거즈 (한국시리즈 직행)
2위: 삼성 라이온즈 (준플레이오프)
3위: LG 트윈스 (준플레이오프) 
4위: 두산 베어스 (와일드카드)
5위: KT 위즈 (와일드카드)
```

### 와일드카드 결정전
```
두산 vs KT (단판승부)
→ 승리팀이 준플레이오프 진출
→ 패배팀은 시즌 종료
```

### 플레이오프 4승 시스템
```
KIA(1위) vs 준플레이오프 승리팀
→ KIA가 1승 어드밴티지로 시작
→ 실제로는 3승만 더 올리면 한국시리즈 진출
```

### 한국시리즈 홈 어드밴티지
```
정규시즌 1위팀이 직행한 경우:
→ 1위팀이 홈 어드밴티지 (4경기 홈)

플레이오프를 통해 올라온 경우:
→ 정규시즌 성적 우수팀이 홈 어드밴티지
```
