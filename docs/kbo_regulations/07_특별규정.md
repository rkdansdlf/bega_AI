# KBO 특별 규정

## 개요
코로나19, 기상 이변, 특수 상황 등 예외적 상황에 대한 특별 규정을 다룹니다.

## 세부 조항

### 조항 1: 코로나19 대응 규정 (2020-2022)
- **무관중 경기**: 방역 지침에 따른 무관중 또는 제한 관중
- **선수 격리**: 확진자 발생 시 밀접 접촉자 격리
- **경기 연기**: 확진자 발생 시 경기 연기 및 일정 조정
- **방역 수칙**: 마스크 착용, 사회적 거리두기 등

### 조항 2: 우천 및 기상 특별 규정
- **태풍 대비**: 태풍 경보 시 경기 사전 연기
- **폭염 대응**: 체감온도 35도 이상시 경기 시간 조정
- **한파 대응**: 영하 10도 이하 시 경기 연기 검토
- **미세먼지**: 매우나쁨 단계 시 선수 건강 고려

### 조항 3: 비상 상황 대응
- **정전 사고**: 조명 시설 고장 시 경기 중단 및 연기
- **시설 파손**: 경기장 시설 심각 손상 시 대체 구장 사용
- **보안 위협**: 테러 위협 등 보안 문제 시 경기 취소
- **자연재해**: 지진, 화재 등 자연재해 시 즉시 대피

### 조항 4: 경기 일정 조정 규정
- **더블헤더 제한**: 주말 더블헤더 최대 월 2회
- **연속 경기**: 팀당 최대 연속 20경기 제한
- **휴식일**: 월 최소 4일 휴식일 보장
- **시즌 연장**: 불가피한 경우 시즌 연장 가능

### 조항 5: 선수 보호 규정
- **연투 제한**: 투수 연속 등판 제한 (3일 이상 간격 권장)
- **타구 보호**: 타구에 맞은 선수 즉시 의료진 검진
- **충돌 예방**: 홈플레이트 충돌 금지 규정
- **뇌진탕 프로토콜**: 뇌진탕 의심 시 즉시 경기 제외

### 조항 6: 기술 장비 오류 대응
- **전광판 오류**: 점수 표기 오류 시 즉시 수정
- **비디오 판독 시스템**: 시스템 오류 시 심판 육안 판정
- **통신 장비**: 코치-선수 간 통신 장비 고장 시 수신호 사용
- **방송 장비**: 중계 장비 문제가 경기에 영향 주지 않음

### 조항 7: 관중 관련 특별 규정
- **관중 제한**: 특별 상황 시 관중 수 제한 또는 금지
- **응원 제한**: 메가폰, 악기 등 응원 도구 제한
- **출입 통제**: 발열 체크, 신분 확인 등 출입 통제
- **비상 대피**: 긴급 상황 시 관중 안전 대피 절차

### 조항 8: 국제 대회 대응
- **선수 차출**: 국가대표 차출 시 대체 선수 등록
- **경기 일정**: 국제 대회 기간 중 경기 일정 조정
- **시설 대여**: 국제 대회를 위한 구장 대여 허용
- **규정 특례**: 국제 대회 관련 특별 규정 적용

## 예시

### 코로나19 대응 사례 (2020년)
```
시즌 개막 연기: 3월 → 5월로 연기
경기 수 축소: 144경기 → 144경기 유지하되 일정 압축
무관중 경기: 시즌 초반 2개월간 무관중
제한 관중: 단계별로 25% → 50% → 100% 개방
```

### 태풍 대응 사례
```
태풍 경보 발령 시:
→ 해당 지역 경기 사전 연기
→ 선수단 안전 확보
→ 구장 시설 보호 조치
→ 연기 경기 일정 재조정
```

### 시설 파손 대체 구장 사례
```
잠실구장 조명탑 고장 (가상)
→ 긴급히 고척돔으로 경기장 변경
→ 관중 티켓 환불 및 재발매
→ 방송 중계 시설 긴급 이설
```

### 선수 보호 적용 사례
```
홈플레이트 충돌 상황:
→ 주자는 슬라이딩으로 홈 터치
→ 포수는 베이스 라인 막기 금지
→ 위반 시 세이프 판정

뇌진탕 프로토콜:
→ 의심 증상 시 즉시 교체
→ 전문의 검진 후 복귀 결정
```
