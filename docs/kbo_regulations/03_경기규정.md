# KBO 경기 규정

## 개요
야구 경기 진행에 관한 세부 규칙과 KBO 특별 규정을 다룹니다.

## 세부 조항

### 조항 1: 경기 진행 규칙
- **투구 시계**: 주자 없을 때 15초, 주자 있을 때 20초
- **타임 요청**: 각 팀 타임아웃 횟수 제한 없음 (합리적 범위)
- **투수 교체**: 한 타자 상대 원칙 (부상 등 예외)
- **대주자**: 9회 이후 주자로만 사용 가능한 선수 지정

### 조항 2: 방해와 보크
- **타자 방해**: 포수가 타자의 스윙을 방해하는 행위
- **주루 방해**: 야수가 공 없이 주자를 방해하는 행위
- **관중 방해**: 관중이 인플레이 볼에 개입하는 경우
- **보크**: 투수의 불법 투구 동작 (1루 진루)

### 조항 3: 홈런 규정
- **인정 홈런**: 담장을 넘거나 파울폴 위로 넘어간 공
- **그라운드룰 더블**: 바운드 후 담장 넘어간 경우
- **관중 방해 홈런**: 관중이 홈런성 타구를 잡은 경우 홈런 인정
- **비디오 판독 대상**: 홈런/파울 경계, 방해 여부 등

### 조항 4: 몸에 맞는 공(HBP)
- **자동 1루 진루**: 투구가 타자의 몸에 맞은 경우
- **회피 의무**: 타자는 합리적으로 피하려고 노력해야 함
- **고의 사구**: 심판 판단으로 투수 퇴장 및 경고
- **보호 장비**: 타자는 헬멧 착용 의무

### 조항 5: 인필드 플라이
- **적용 상황**: 무사/1사, 1·2루 또는 만루 상황
- **타구 조건**: 내야 뜬공으로 일반적인 수비 노력으로 잡을 수 있는 공
- **자동 아웃**: 타자 즉시 아웃, 주자는 리드 가능
- **심판 콜**: "인필드 플라이" 선언 필수

### 조항 6: 우천 및 기상 규정
- **경기 중단**: 비, 번개, 강풍 등으로 인한 중단
- **재개 조건**: 기상 상황 호전 및 구장 상태 회복
- **콜드게임**: 5이닝 이상 진행 후 기상으로 인한 조기 종료
- **더블헤더**: 연기 경기 소화를 위한 같은 날 2경기 진행

### 조항 7: 덕아웃 규정
- **입장 허용**: 등록선수, 코치진, 구단 관계자만 입장
- **응원**: 덕아웃에서 상대팀 비방 행위 금지
- **장비**: 헬멧, 배트 등은 정리정돈 필수
- **의료진**: 팀 의무 트레이너 벤치 입장 허용

## 예시

### 투구 시계 적용
```
주자 없음: 투수는 15초 내 투구 
→ 위반 시 볼 선언

주자 있음: 투수는 20초 내 투구
→ 위반 시 볼 선언
```

### 인필드 플라이 상황
```
1사 1·2루 상황에서 3루수 앞 뜬공
→ 심판 "인필드 플라이" 선언
→ 타자 즉시 아웃, 주자는 태그업 가능
```

### 보크 주요 사례
```
- 투구 동작 중 멈추기
- 1루 견제 시 발을 투수판에서 빼지 않음
- 투구판을 딛고 1루 이외로 던지기
- 투수판 위에서 공을 떨어뜨리기
```

### 그라운드룰 더블
```
타구가 바운드 후 담장을 넘음
→ 타자는 2루까지만 진루
→ 주자들도 2개 베이스만 진루
```
