# KBO 기술 규정

## 개요
경기 기록, 통계, 심판 운영 등 기술적 사항에 관한 규정을 다룹니다.

## 세부 조항

### 조항 1: 기록 규정
- **공식 기록**: KBO 공인 기록원이 작성한 기록만 공식 인정
- **기록 정정**: 경기 종료 후 24시간 이내 정정 신청 가능
- **개인 기록**: 타격, 투구, 수비 기록을 종합하여 관리
- **팀 기록**: 승부, 득점, 타격 등 팀 단위 기록 관리

### 조항 2: 안타와 실책 판정
- **안타 기준**: 타자가 안전하게 베이스에 도달한 타구
- **실책 기준**: 일반적인 수비 노력으로 아웃시킬 수 있었으나 실패
- **기록위원 판단**: 애매한 상황은 기록위원이 최종 판단
- **실책 종류**: 포일, 송구 실책, 낙구 등

### 조항 3: 투구 기록
- **승리 투수**: 선발은 5이닝, 구원은 결정타 허용 후 승리팀이 추가 득점
- **패전 투수**: 결정적인 실점을 허용한 투수
- **세이브**: 9회 이후 3점차 이하에서 경기를 마감한 구원투수
- **홀드**: 세이브 상황에서 리드를 유지하며 강판된 구원투수

### 조항 4: 통계 산출 규정
- **타율**: 안타수 ÷ 타수 (볼넷, 몸에 맞는 공 제외)
- **출루율**: (안타+볼넷+몸에 맞는 공) ÷ (타수+볼넷+몸에 맞는 공+희생플라이)
- **장타율**: 총 루타수 ÷ 타수
- **방어율**: (자책점 × 9) ÷ 투구 이닝수

### 조항 5: 심판 운영
- **심판진 구성**: 주심 1명, 루심 3명 (총 4명)
- **비디오 판독**: 특정 상황에서 영상 판독 실시
- **항의 절차**: 감독만 심판에게 항의 가능
- **퇴장 권한**: 심판은 선수, 코치 퇴장 권한 보유

### 조항 6: 경기 용품 규정
- **공인구**: KBO 공인 경기구만 사용
- **배트 규격**: 길이 106.7cm 이하, 직경 7cm 이하
- **글러브**: 포지션별 크기 제한 존재
- **유니폼**: KBO 승인된 디자인과 번호 사용

### 조항 7: 기록실 운영
- **기록원**: 경기당 3명의 공식 기록원 배치
- **실시간 기록**: 타석별, 이닝별 상세 기록 작성
- **데이터 전송**: 경기 종료 즉시 KBO 본부로 전송
- **보관 기간**: 모든 경기 기록 영구 보관

## 예시

### 승리 투수 판정 사례
```
선발투수: 5이닝 3실점으로 강판 (팀이 5-3 리드)
→ 이후 팀이 계속 리드 유지하여 승리
→ 선발투수가 승리투수

구원투수: 8회에 등판하여 동점 상황을 만듦
→ 9회에 팀이 결승점 득점하여 승리
→ 구원투수가 승리투수
```

### 세이브 기록 조건
```
다음 중 하나 충족 시 세이브 기록:
1. 3점차 이하에서 3이닝 이상 투구
2. 타이잉런이 베이스에 있거나 타석에 들어설 상황
3. 1점차에서 1이닝 이상 투구
```

### 안타와 실책 판정
```
3루수 앞 땅볼: 평범한 수비로 아웃 가능
→ 실책 처리

외야 라인드라이브: 다이빙캐치 실패  
→ 안타 처리 (난이도 높은 수비)
```

### 기록 정정 사례
```
경기 중 실책으로 기록된 플레이
→ 경기 후 재검토 결과 안타로 정정
→ 24시간 이내 정정 신청 및 승인
```
